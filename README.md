# Advent of Code 2025

![Stars](https://img.shields.io/badge/stars-16%2A-gold?logo=adventofcode) ![AI](https://img.shields.io/badge/clanker%20code-0%25-green?logo=claude)

To run use the following command:
```powershell
dotnet run --project Ato.AoC2025 [day##]
```

`day##` is optional to skip console ui and directly print the result of day number ##. eg. day01.
